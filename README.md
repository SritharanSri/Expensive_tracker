# Spendly - Expense Tracker & Budget

Spendly is a modern, feature-rich web application designed to help users track their expenses, manage budgets, and achieve their financial goals.

## Features

*   **Expense & Income Tracking:** Easily record and categorize your daily transactions.
*   **Budget Management:** Set savings goals and track your progress.
*   **AI Receipt Scanner:** Quickly add expenses by scanning your receipts using our Gemini AI-powered scanner.
*   **Premium Options:** Unlock advanced features, including unlimited AI scans and an ad-free experience, with our Google Play Billing integrated premium tier.
*   **Freemium Features:** Free users can access AI tools with a limited usage allowance, while enjoying standard ad-supported functionalities.
*   **Responsive Design:** A beautiful, responsive user interface built for all devices.

## Tech Stack

*   **Frontend:** Next.js, React, TypeScript, Tailwind CSS
*   **AI Integration:** Google Gemini ABI
*   **Monetization:** Google Play Billing (simulation) & Google AdSense

## Getting Started

### Prerequisites
*   Node.js installed

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/SritharanSri/Expensive_tracker.git
   ```
2. Navigate to the project directory:
   ```bash
   cd "Expensive_tracker/Frontend"
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and navigate to `http://localhost:3000`.
